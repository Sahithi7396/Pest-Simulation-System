🌾 Rythu Mitra – Pest Spread Prediction System

Rythu Mitra is a smart agriculture-based web application designed to predict pest spread across farmland before major crop damage occurs.

Built during the EleXcentra 2026 Hackathon, the project focuses on helping farmers take preventive action instead of reacting after infestation spreads.

🚀 Problem Statement

Pest infestations spread rapidly across agricultural land, often causing severe crop damage before farmers can identify the issue.

Most existing agricultural tools focus on:

Detecting diseases after infection
Suggesting pesticides after damage begins

Rythu Mitra explores a proactive approach:

Can we predict pest spread early and help farmers act before crops are affected?

💡 Solution

Rythu Mitra simulates pest propagation using a Cellular Automata-based grid system.

Each grid cell represents a section of farmland, and pest spread is simulated using the Von Neumann Neighborhood Model, where each cell interacts with its adjacent neighbors:

The system dynamically predicts spread patterns based on environmental conditions.
